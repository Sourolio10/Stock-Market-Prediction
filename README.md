# Stock-Market-Prediction
This program is predicting the stock price of Amazon Stock Data. By this program, we can conclude that Linear Regression package of Machine Learning in Python can be used effectively to predict the stock market prices of different stock data-sets. Linear Regression is a machine learning algorithm based on supervised learning. It performs a regression task. Regression models a target prediction value based on independent variables. It is mostly used for finding out the relationship between variables and forecasting.
